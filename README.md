# Ma Matzavinu

A simple, mobile-first Hebrew expense tracking app built with Python and Flask. Designed for fast and clean family budgeting, with an RTL interface optimized for phone use.

## Live App
https://ma-matsavinu.onrender.com/

## Features
- View expenses  
- Add new expenses  
- Edit and delete entries  
- SQLite storage  
- Clean RTL UI  
- Automatic deployment via Render

## Tech Stack
- Python + Flask  
- HTML (Jinja templates)  
- CSS (mobile-first)  
- SQLite database  
- Render deployment  
- GitHub version control

## Run Locally
```bash
git clone https://github.com/nanoo26/ma-matsavinu.git
cd ma-matsavinu
pip install -r requirements.txt
python app.py
